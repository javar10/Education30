# :link: [Letter Tracing](https://javar10.github.io/Education30/08%20-%20Letter%20Tracing/)

The Letter Tracing app if a fun way for students to practice forming letters. 
The user can clear the screen and try a letter again with the "Try Again" button.
A new letter will be generated with the "New Letter" button. 
This app does not currently work on mobile devices.
