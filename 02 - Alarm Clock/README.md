# [Alarm Clock](https://javar10.github.io/Education30/02%20-%20Alarm%20Clock/)

The Alarm Clock app allows the user to store a schedule and then set times throughout the day for different alarms to sound. 
The visual timer is displayed so young students have a sense of how much time they have left before they will be asked to transition to something new. 
This appears on an analog clock so that students can gain a better sense of how a clock works and get a feel for how long one minute, five minutes, thirty minutes, etc. really is.
