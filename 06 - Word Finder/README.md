# :link: [Word Finder](https://javar10.github.io/Education30/06%20-%20Word%20Finder/)

With Word Finder, the user can search for words that are a certain number of letters and contain specific spelling patterns.
The app will then display a list of these words, highlighting the part of the word that was searched for. 
As the user input changes, the list will also change dynamically.
